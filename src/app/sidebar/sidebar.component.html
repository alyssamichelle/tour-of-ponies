Heatmap Component

Charts › Series Types › Heatmap

 ng add @progress/kendo-angular-charts 

Heatmap charts use colors to indicate the relative value of data points in two dimensions. Heatmap charts, also known as "cluster heat map", are suitable for visualizing the magnitude of a value over two dimensions. The X and Y values are discrete and can be set in any order.

Skeleton Component

Indicators › Skeleton

 ng add @progress/kendo-angular-indicators 

The Skeleton component helps developers to speed up the initial page load time by displaying a simplified preview of the content. This preview acts as a placeholder for the content while it loads. At the same time, the user will get a general idea about what will appear on the page.

CircularGauge Component

Gauges › CircularGauge

 ng add @progress/kendo-angular-gauges 

The Circular Gauge represents a value using an arc over a fully enclosed circle.

This gauge type is very similar to the Arc Gauge and supports a similar set of features.

Compared to the Radial Gauge, this gauge uses a colored arc instead of an arrow pointer. This leaves the center free for a user-defined Center Template.
